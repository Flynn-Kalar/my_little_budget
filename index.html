개인정보처리방침

최종 수정일: 2026-06-08

나작부(My Little Budget)는 사용자의 개인정보를 수집하거나 개발자가 운영하는 서버로 전송하지 않습니다.

1. 수집하는 정보
* 개인정보 수집 없음
* 회원가입 없음
* 이메일 수집 없음
* 광고 식별자 수집 없음
* 위치 정보 수집 없음

2. 데이터 저장 위치
* 모든 데이터(가계부, 예산, 투자 기록 등)는 사용자 기기 내부에 저장됩니다.
* 개발자가 운영하는 외부 서버로 전송되지 않습니다.

3. 백업 및 복원
* 사용자가 직접 내보낸 백업 파일(JSON)은 사용자 기기에 저장됩니다.
* 개발자는 백업 파일 및 사용자 데이터에 접근하지 않습니다.

4. 제3자 제공
* 사용자 데이터를 제3자에게 제공하지 않습니다.

5. 개인정보처리방침 변경
* 기능 변경 또는 법령 변경이 있는 경우 본 방침이 수정될 수 있습니다.
