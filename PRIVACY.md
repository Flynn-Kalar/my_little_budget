# 개인정보처리방침

my_little_budget는 개인 예산 관리를 위한 로컬 우선 앱입니다.

## 수집하는 정보

앱은 사용자가 직접 입력한 자산, 거래, 예산, 투자, 메모 데이터를 앱 내부 저장소에 보관합니다.

## 데이터 저장

기본 데이터는 사용자의 기기 안에 저장됩니다. 사용자가 백업 또는 동기화 기능을 설정한 경우, 사용자가 입력한 외부 저장소 설정에 따라 백업 데이터가 저장될 수 있습니다.

## 제3자 제공

앱은 사용자의 데이터를 임의로 제3자에게 판매하거나 제공하지 않습니다.

## 문의

문의와 개선 요청은 GitHub 저장소 이슈를 통해 남길 수 있습니다.
